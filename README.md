# Фрактальное дерево

## Описание

Реализация фрактального дерева с помощью рекурсивного алгоритма и встроенной библиотеки `turtle` в Python.

Программа строит геометрическую фигуру, где каждая ветка рекурсивно разделяется на две меньшие ветки, формируя структуру, напоминающую дерево. Цвета веток циклически меняются из заданной палитры.

## Как запустить

1. Убедитесь, что установлен **Python 3.6+**.
